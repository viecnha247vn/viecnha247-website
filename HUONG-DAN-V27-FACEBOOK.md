2. ✅ Vào **Cài đặt → Website** → nhập `https://viecnha247.vn`
3. ✅ Đăng bài đầu tiên (dùng đoạn giới thiệu đã viết hôm trước)
4. ✅ Mời 5-10 bạn bè thích Page

### Trên website (sau upload):
- Tạo file `og-image.jpg` (1200×630px) — logo + slogan
- Upload vào root folder
- Khi share Facebook sẽ thấy preview đẹp
